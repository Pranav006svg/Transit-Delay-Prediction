# Transit Delay Prediction

A modular smart transit tracking application that provides live updates, optimal route comparisons, and intelligent estimations of travel durations and potential delays.

## Features
- **Live Transit Tracking**: Monitor buses, trains, and metro movement via real-time data integration.
- **Smart Prediction**: Calculates travel times and highlights likely delays using underlying transit analysis models.
- **Route Optimization**: Compares options (bus, train, walking) so you can find the most reliable route based on predicted metrics.
- **Dashboard Ecosystem**: An analytical suite providing performance statistics, on-time percentage updates, and incident visualizations.

## Getting Started

### Prerequisites

Ensure you have Node.js installed to serve the application locally.

### Installation

1. Install all dependencies using `npm`:
```sh
npm install
```

2. Run the development environment:
```sh
npm run dev
```

The application should be accessible via your localhost on port `8080`.

## Scripts

- `npm run dev` - Launch the local development server (Vite).
- `npm run build` - Create the production-ready build of the application.
- `npm run lint` - Trigger `eslint` to verify code quality.
- `npm run test` - Execute available test suites via `vitest`.
